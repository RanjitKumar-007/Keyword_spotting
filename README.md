Keyword_spotting in audio:

This project implements an end-to-end multilingual audio threat detection pipeline that automatically processes audio recordings, converts speech into text, translates non-English content into English, and identifies potential threats using keyword-based analysis. The system begins by ingesting audio files in formats such as WAV, MP3, and FLAC, followed by preprocessing steps including noise reduction, resampling to 16 kHz, and voice activity detection (VAD) to remove silence and improve transcription quality. The cleaned audio is then passed to a language detection module powered by Faster-Whisper Large-v3, which determines the spoken language using an initial segment of the audio. If the detected language is English, the audio is transcribed directly using Faster-Whisper Large-v3. For Indian languages such as Hindi, Telugu, Tamil, Kannada, Malayalam, Bengali, Gujarati, Marathi, Punjabi, and Urdu, transcription is performed using AI4Bharat IndicConformer, and the resulting transcript is translated into English using AI4Bharat IndicTrans2. This ensures that all downstream analysis operates on a common English representation regardless of the input language.

The generated transcript undergoes a text normalization process that includes lowercase conversion, punctuation removal, whitespace cleanup, Unicode normalization, and format standardization to improve consistency and matching accuracy. The normalized text is then analyzed using a configurable keyword and synonym detection engine that compares the transcript against a threat dictionary containing security-related terms, aliases, and synonyms. Detected keywords are counted and used to evaluate the potential threat level of the audio content. The system generates detailed logs and reports in JSON or text format, capturing information such as the input audio file, detected language, transcription model used, original and translated transcripts, normalized text, detected keywords, and threat analysis results. A dedicated keyword dictionary management component allows users to dynamically add, modify, or remove keywords, synonyms, and rules without requiring code changes. Built using Python, PyTorch, Faster-Whisper, AI4Bharat IndicConformer, IndicTrans2, and efficient keyword-matching libraries such as RapidFuzz and FlashText, this framework provides a scalable, configurable, and production-ready solution for multilingual audio intelligence, surveillance, and automated threat detection.
